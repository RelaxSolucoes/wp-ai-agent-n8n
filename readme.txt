Plugin WordPress para integração com Agente de IA via N8N e Evolution API.

== Description ==

O **WP AI Agent N8N** é um plugin WordPress profissional que permite integrar seu site com um agente de IA configurado no N8N. Ele oferece uma solução completa para automação de atendimento e integração com WhatsApp.

= Funcionalidades Principais =

* **Widget de Chat**: Chat interativo que se integra com seu fluxo N8N
* **Formulário de Contato**: Formulário que envia mensagens para o N8N
* **Integração Evolution API**: Conexão com WhatsApp via Evolution API
* **Interface Administrativa**: Painel de configuração intuitivo e organizado
* **Shortcodes Flexíveis**: Inserção fácil em qualquer página ou post
* **Design Responsivo**: Funciona perfeitamente em dispositivos móveis
* **Sistema de Validação**: Validação robusta de formulários
* **Segurança**: Nonces, sanitização e verificação de permissões

= Casos de Uso =

* Atendimento automatizado via chat
* Formulários de contato inteligentes
* Integração com fluxos de automação N8N
* Conexão com WhatsApp via Evolution API
* Chatbots personalizados para sites
* Automação de respostas a clientes

= Integração com N8N =

O plugin envia dados estruturados para o N8N, permitindo:
* Processamento inteligente de mensagens
* Roteamento automático de respostas
* Integração com APIs de IA (OpenAI, Claude, etc.)
* Fluxos de automação complexos
* Análise de dados de usuários

= Compatibilidade =

* WordPress 5.8+
* PHP 7.4+
* N8N 0.200+
* Evolution API 2.0+
* Todos os temas modernos
* Navegadores modernos

== Installation ==

= Instalação Automática (Recomendado) =

1. Acesse o painel administrativo do WordPress
2. Vá para Plugins > Adicionar Novo
3. Clique em "Enviar Plugin"
4. Selecione o arquivo ZIP do plugin
5. Clique em "Instalar Agora"
6. Ative o plugin

= Instalação Manual =

1. Faça upload da pasta `wp-ai-agent-n8n` para o diretório `/wp-content/plugins/` do seu WordPress
2. Ative o plugin através do menu 'Plugins' no WordPress

= Configuração Inicial =

1. Acesse o menu **AI Agent N8N** no painel administrativo
2. Configure as conexões na aba **Conexão**
3. Ative as funcionalidades desejadas nas abas correspondentes
